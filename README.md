# Mecanic OS

Sistema premium de gestión de talleres mecánicos y facturación electrónica para El Salvador. Basado en el esquema de base de datos de Grupo Gema.

## Estructura del Proyecto
- `/src`: Código fuente del frontend y backend.
- `/docs`: Documentación técnica y análisis.
- `/scratch`: Scripts y utilidades de desarrollo.
